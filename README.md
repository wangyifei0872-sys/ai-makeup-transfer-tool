# AI Makeup Transfer Tool

本地 Next.js 第一版 mock 页面，用于演示 AI 妆容迁移工具流程。

## 功能

- 上传原始插画、妆容参考图和可选遮罩图
- 上传后本地预览，支持重新上传和移除
- 设置妆容强度、修改范围、保留原图程度、输出数量
- 固定展示当前模型：Nano Banana 2
- 固定模型 ID：gemini-3.1-flash-image
- 使用 mock loading 和原始插画占位结果模拟生成流程
- 支持重新生成、继续优化生成和下载结果图

## 安装依赖

```bash
npm install
```

## 启动本地网页

```bash
npm run dev
```

## 本地访问地址

```text
http://localhost:3000
```
